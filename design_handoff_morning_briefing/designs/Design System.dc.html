The foundation

A calm, editorial system for daily intelligence.

Crisp white paper, ink-black type set in Spectral, and a single deep-blue accent — layered over translucent liquid-glass surfaces. Every token, type style and component below is reused verbatim across the landing page, setup wizard, briefing viewer and archive.
